# Unity_miningsimulator
Данный симулятор является дипломным проектом - симулятором для горнотехнической промышленности по обучению шахтеров. Представляет собой проект на Unity (в данный момент - до версии 5.х). Компилируется, однако скомпилированного варианта внутри не присутствует. Желательно уведомить владельца о взятии файлов из репозитория.

Текущая версия обладает рядом свойств.
* вид от первого дица
* анимации объектов шахты
* наличие триггеров для вызова анимации
* здоровье игрока, которое уменьшается при падении
* выносливость игрока (продолжительность бега)
* скрипт фонарика, который включается и выключается по клавише
* скрипт звука шагов
* освещение шахты
* отдельные шейдеры для улучшения качества изображения
* миникарта с условными обозначениями объектов на ней
* легенда к миникарте с подписями, условными обозначениями и надписями
